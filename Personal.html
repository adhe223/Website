Faolin: Text Based RPG

Faolin is a text based RPG that I put together in C++ as a learning exercise. I was hoping to gain a greater understanding of game principles and structure. The game features an algorithm to randomly generate the world (mountains, rivers, plains), a working map available to the player, a leveling system, the ability to save and load games, as well as other general RPG elements.

Minesweeper

This is a text based minesweeper game. The game is completely functional and features a working high score list. Written in C++ using VS10.
