WikiLocale: Android App

WikiLocale is my first foray into mobile development. The goal of the project was to make an app that is useful for me and in the process build valuable, mobile development skills as well as accrue more experience in Java. The app detects when the user is traveling away from home and sends them a notification that links to the Wikipedia page of the city they are currently in. It tests for various situations where it won't fire a notification based on the user's home location (user defineable) or distance traveled since last update. Written in Java using Eclipse and Google's Android SDK.

Faolin: Text Based RPG

Faolin is a text based RPG that I put together in C++ as a learning exercise. I was hoping to gain a greater understanding of game principles and structure. The game features an algorithm to randomly generate the world (mountains, rivers, plains), a working map available to the player, an inventory system, leveling attributes, the ability to save and load games, as well as other essential RPG elements. The project was written in C++ using VS10.

Minesweeper

This is a text based minesweeper game. The game is completely functional and features a working high score list. This was originally a coding assignment through school. We had four such projects that semester, this was the largest of the four and definitely the one I enjoyed the most. Features include a text based display, the functioning high scores list, customizable grids, random mine placement, as well as a recursive algorithm to check for the presence of mines. Project is written in C++ using VS10.
